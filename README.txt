Project Writeup:
Tech Stack: MongoDB, Node.js, Express.js, React

This is a full-stack CRUD web application. You can create a user as well as associated characters for that user. 

NEED TO INSTALL: 

node.js: https://nodejs.org/en

HOW TO RUN

FRONTEND:

1. cd frontend

2. run "npm start"

BACKEND: 

1. cd backend

2. run "npm start"
